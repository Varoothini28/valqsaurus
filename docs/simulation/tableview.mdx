## 1. Accessing Table View

There are two ways to switch to the Table View in the Simulate screen:

1. By clicking on the Switch View icon next to the simulation bar at the top 

2. By clicking on the Table option in the Visualization section in the Navigation Panel. 

<img alt="Accessing Table View" src={useBaseUrl('/doc-images//tvv-1.png')} />

On collapsing the Navigation Panel, you will see the following screen. There is a Search option 
on the left, Settings on the top right and the Table occupies the 
rest of the screen.

<img alt="Table View" src={useBaseUrl('/doc-images//tvv-5.png')} />

## 2. Search option

You can type the node you want to search and select the same from the dropdown.

<img alt="Search" src={useBaseUrl('/doc-images//tvv-6.png')} />

The selected node and its descendants will be displayed.

<img alt="Search" src={useBaseUrl('/doc-images//tvv-7.png')} />

## 3. Summarized View

By default, the table is shown in the summarized mode. 

- When there are no simulations, aggregated baseline 
and comparison series will be displayed.

<img alt="Summarized View" src={useBaseUrl('/doc-images//tvv-9.png')} />

- When there are simulations, original and simulated baseline, comparison values and 
simulation impact will be displayed.

<img alt="Settings" src={useBaseUrl('/doc-images//tvv-10.png')} />

## 4. Period View

On selecting the period view from the settings, you'll be able to view the table by periods.

<img alt="Period View" src={useBaseUrl('/doc-images//tvv-4.png')} />

## 5. Customize Columns

By clicking on settings, you can see the following screen.

<img alt="Settings" src={useBaseUrl('/doc-images//tvv-11.png')} width = "50%" height = "50%" />

In the left, you can search for columns, select or unselect them based on whether you want it to be displayed.

In the right, you can reorder columns by dragging and dropping the fields.
